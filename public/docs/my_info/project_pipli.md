# Pipli - Retailer Portal

**Role:** Frontend Developer

## Description
Developed a responsive retailer portal web application using React, enabling retailers to manage feedback, e-bills, analytics, business/store/POS systems, bill design, and promotions. Implemented features for feedback management, e-bill management, analytics, business/store/POS management, bill design customization, and promotion management. Ensured performance, scalability, and security through optimization techniques and authentication/authorization features.

## Responsibilities
- Developing a responsive web application using NextJs, HTML, CSS, and JavaScript, with a focus on creating an intuitive and user-friendly interface for retailers to manage their stores and businesses.
- Integrating feedback management features such as survey creation, response tracking, and sentiment analysis, using technologies such as Google Forms.
- Implementing e-bill management features, including invoice creation, E bill listing, and tracking.
- Designing and implementing analytics features that allow retailers to track and analyze sales data, customer behavior, and inventory management, using tools such as Google Analytics.
- Developing business, store, and POS systems management features, such as inventory management, customer management and customer feedback management.
- Designing and implementing bill design features, including customization of bill templates, adding logos or branding, and support for multiple currencies and languages.
- Developing promotion management features, such as creating and managing discount codes, loyalty programs, and referral campaigns.
- Implementing authentication and authorization features, such as user roles and permissions, using technologies such as OAuth or JWT.
- Ensuring the web application is optimized for performance, scalability, and security, using techniques such as code splitting, caching.
- Collaborating with other developers, designers, and project managers to ensure the web application meets business requirements, design specifications, and timelines.

## Technologies
HTML5, CSS3, JavaScript, React, Prime React, REDUX, Prime React charts, AWS S3, Google Forms, Google Analytics, OAuth, JWT, Adobe XD.
